# stock-dashboard-dash

路	Built and deployed an interactive stock visualization web dashboard that fetches, processes and displays stock price and financial data with chart.

路	The dashboard app is useful for long-term investors who seek due diligence in stocks based on fundamentals (income statement, balance-sheet, etc.). 

路	The dashboard app is also useful for short-term investors who seek to find appropriate entry/exit  points based on correlation between stock price and sensitive indicators such as RSI, KDJ, W%R, etc.

路	Tools: Python, Pandas, Dash and HTML/CSS.
